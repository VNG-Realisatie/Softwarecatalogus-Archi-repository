Componentencatalogus model ()
Kennismodel van indelingen voor het vindbaar maken van oplossingen in de componentencatalogus. Bij voorkeur worden de indelingen overgenomen uit de GEMMA. Nog te bepalen wat als uitbreiding uitgewerkt gaat worden in de GEMMA en wat specifiek voor geldt voor het componentencatalogus model Daarnaast bevat de componentencatalogus de GEMMA standaardenlijst waarmee leverancier kunnen registreren welke standaarden en API-standaarden zij ondersteunen. De API's en API standaarden gaan als uitbreiding opgenomen worden in de GEMMA
