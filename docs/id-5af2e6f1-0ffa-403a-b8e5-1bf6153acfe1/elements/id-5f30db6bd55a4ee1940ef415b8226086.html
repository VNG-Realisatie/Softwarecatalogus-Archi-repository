Open Registers ()
A Nextcloud-based app for managing structured, authoritative datasets (registers). It ensures versioned, trusted, and structured data storage, accessible via API and UI. Key Features: - Store and manage registers in JSON, CSV, or XML format. - Version control and access permissions. - WebDAV & API integration for external system access. - Federation support for syncing across Nextcloud instances.
