ArchiMate File Format ()
Met het ArchiMate Bestandsuitwisselformaat kunnen architectuurtools ArchiMate architectuurmodellen met elkaar uitwisselen. Deze standaard wordt ook aangeduid als ArchiMate Model Exchange File Format (AMEFF)
ToelichtingArchitectuurtools kunnen een architectuurmodel exporteren naar het AMEFF uitwisselformaat. Dit exportbestand kan vervolgens worden geïmporteerd in een andere tool. De architectuurtool Archi (open source) is een voorbeeld van een architectuur met ondersteuning voor het AMEFF bestandsformaat. (https://www.archimatetool.com)
