CI/CD-tool ()
Tool die het bouwen, testen (CI) en uitrollen (CD) van code naar omgevingen automatiseert en coördineert.
ToelichtingDe Continuous Integration/Continuous Delivery-tool coördineert en automatiseert het proces door andere tools aan te sturen voor bouwen, containerisatie en uitrol, maar voert deze stappen niet zelf uit. Voorbeelden zijn GitHub Actions, GitLab CI/CD en Jenkins.
