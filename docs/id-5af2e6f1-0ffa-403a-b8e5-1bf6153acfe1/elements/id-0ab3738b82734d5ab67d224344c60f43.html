Applicatieversie (Voorzieningversie) ()
De eigenaar van de applicatie registreert minimaal één applicatieversie. De 'actuele versie' wordt als voorkeuze getoond. De aanbieder van een dienst registreert voor welke applicatie een dienst aangeboden wordt. Als geen versie wordt opgegeven geldt dit impliciet voor alle versies De gebruiker van een dienst registreert welke applicatie gebruikt wordt. Het registreren van een versie is optioneel.
